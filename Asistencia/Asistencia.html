def est1(x):
    return np.mean(x)

def est2(x):
    n = len(x)
    return np.sum((x - np.mean(x))**2) / n

def est3(x):
    n = len(x)
    return -0.5 + np.sqrt(0.25 + np.sum(x**2) / n)

Bootstrap no paramtrico

Para esta ayudanta, nos basamos en el libro de Wasserman[^1].

Haremos bootstrap no paramtrico para estimar los cuantiles 0.025 y 0.975 de una v.a. exponencial.

Sea $X\sim \mathrm{exp}(\lambda = 5)$. Iniciamos con una muestra aleatoria de $X$ con $n=300$.

[^1]: Larry Wasserman, All of Statistics, 2004

In[]:
# Bootstrap no paramtrico. Parmetros de la distribucin real
n = 300
lambda_val = 5
x = np.random.exponential(scale=1/lambda_val, size=n)

Para el bootstrap, tomaremos $K=100$ muestras de tamao $m=30$, de las cuales obtendremos el cuantil emprico 0.025 $q_1^k$ y 0.975 $q_2^k$.

r = np.mean(x)

Bootstrap paramtrico

Haremos bootstrap paramtrico para estimar el cuantil 0.025 y 0.975 de una muestra exponencial. Sea $X\sim \mathrm{exp}(\lambda = 5)$. Recordemos \begin{align*} \mathbb{E}[X] &= \frac{1}{\lambda} \end{align*} Por lo que el estimador de $\lambda$ por el mtodo de momentos es: \begin{align} \hat{\lambda} &= \frac{1}{\overline{X}} \end{align} Iniciamos con una muestra aleatoria de $X$ con $n=300$.

In[]:
# Bootstrap no paramtrico. Parmetros de la distribucin real
n = 300
lambda_val = 5
x = np.random.exponential(scale=1/lambda_val, size=n)

Para el bootstrap, tomaremos $K=100$ muestras de tamao $m=30$, de las cuales obtendremos el estimador por el mtodo de momentos de cada muestra $\hat{\lambda}_k$, para posteriormente encontrar los cuantiles terico 0.025 $q_1^k$ y 0.975 $q_2^k$..

In[]:
# Bootstrap paramtrico. Parmetros de bootstrap
r = np.mean(x)
k = 100
m = 30
q1 = 0.025
q2 = 0.975

q1_est = np.zeros(k)
q2_est = np.zeros(k)

# Bootstrap paramtrico. Bootstrap
for i in range(k):
    z = np.random.exponential(scale=1/r, size=m)
    r_boot = np.mean(z)
    q1_est[i] = expon.ppf(q1, scale=1/r_boot)
    q2_est[i] = expon.ppf(q2, scale=1/r_boot)

Finalmente, mostramos la estimacin de la media y la desviacin estndar de los cuantiles $\overline{q_1}$, $\overline{q_2}$, $\mathrm{sd}(q_1)$, $\mathrm{sd}(q_2)$ obtenida por bootstrap paramtrico.

Intervalos de confianza a partir de bootstrap paramtrico

Haremos bootstrap paramtrico para estimar intervalos del 95% de confianza de los cuantiles 0.025 y 0.975 de una muestra exponencial.

Sea $X\sim \mathrm{exp}(\lambda = 5)$. Recordemos \begin{align*} \mathbb{E}[X] &= \frac{1}{\lambda} \end{align*} Por lo que el estimador de $\lambda$ por el mtodo de momentos es: \begin{align} \hat{\lambda} &= \frac{1}{\overline{X}} \end{align} Iniciamos con una muestra aleatoria de $X$ con $n=300$.

In[]:
# Bootstrap no paramtrico. Parmetros de la distribucin real
n = 300
lambda_val = 5
x = np.random.exponential(scale=1/lambda_val, size=n)

Para el bootstrap, tomaremos $K=100$ muestras de tamao $m=30$, de las cuales obtendremos el estimador por el mtodo de momentos de cada muestra $\hat{\lambda}_k$, para posteriormente encontrar los cuantiles terico 0.025 $q_1^k$ y 0.975 $q_2^k$..

In[]:
# Bootstrap paramtrico. Parmetros de bootstrap
r = np.mean(x)
k = 100
m = 30
q1 = 0.025
q2 = 0.975

q1_est = np.zeros(k)
q2_est = np.zeros(k)

# Bootstrap paramtrico. Bootstrap

for i in range(1, k+1):
    z = np.random.exponential(scale=1/r, size=m)
    r_boot = np.mean(z)
    q1_est[i-1] = expon.ppf(q1, scale=1/r_boot)
    q2_est[i-1] = expon.ppf(q2, scale=1/r_boot)

Ahora, mostramos los intervalos de confianza por el mtodo normal de los cuantiles $\overline{q_1}$, $\overline{q_2}$, $\mathrm{sd}(q_1)$, $\mathrm{sd}(q_2)$ obtenidos por bootstrap paramtrico.

In[]:
# Bootstrap paramtrico. Resultados incluyendo intervalos de confianza
Q1 = np.mean(q1_est)
Q2 = np.mean(q2_est)

S1 = np.std(q1_est, ddof=1)  # ddof=1 for unbiased standard deviation
S2 = np.std(q2_est, ddof=1)

T1L = Q1 - S1 * norm.ppf(0.975)
T1R = Q1 + S1 * norm.ppf(0.975)

T2L = Q2 - S2 * norm.ppf(0.975)
T2R = Q2 + S2 * norm.ppf(0.975)

q1_real = expon.ppf(q1, scale=1/lambda_val)
q2_real = expon.ppf(q2, scale=1/lambda_val)

print("Por el mtodo normal:")
print(f"El intervalo de confianza para el cuantil {q1:.3f} es ({T1L:.5f}, {T1R:.5f})")
if T1L < q1_real < T1R:
    print(f"El cuantil {q1:.3f} est en el intervalo de confianza")
else:
    print(f"El cuantil {q1:.3f} no est en el intervalo de confianza")

print(f"El intervalo de confianza para el cuantil {q2:.3f} es ({T2L:.5f}, {T2R:.5f})")
if T2L < q2_real < T2R:
    print(f"El cuantil {q2:.3f} est en el intervalo de confianza")
else:
    print(f"El cuantil {q2:.3f} no est en el intervalo de confianza")
